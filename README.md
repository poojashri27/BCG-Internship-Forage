# BCG-Internship-Forage
I worked on a project to create a customer churn analysis simulation using Python. This project built my data science skills in a real-world context, as well as my presentation skills through creating an executive summary of my findings for the team. 

## Customer Churn Prediction & Driver Analysis
## Overview
I am thrilled to present this advanced predictive modeling project, completed as part of the Boston Consulting Group (BCG) Data Science Virtual Experience Program on Forage.
This project addresses a critical business challenge: customer churn. By engineering robust features and deploying machine learning algorithms, the work bridges the gap between raw consumption data and proactive retention strategies. It highlights strong proficiency in predictive analytics, feature engineering, and strategic problem‑solving, while demonstrating the ability to translate technical outputs into actionable business insights.

* Build predictive models to support client decision-making.
